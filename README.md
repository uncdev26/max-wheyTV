# Max WheyTV

Universal Stremio streaming addon — 1000+ movies & series from every corner of the world.

## Features

- 🔥 Trending, Cinema, Hollywood, Bollywood, South Indian, Asian, Turkish catalogs
- 📺 Top Series, Indian Drama, Anime catalogs
- 🎬 MovieBox streaming backend — fast, multi-language
- ⭐ RPDB rating posters
- 📱 Platform-aware speed optimization (TV/Mobile/Desktop)
- 🌍 60+ languages, 29 countries

## Install

Add this URL to Stremio:

```
https://max-wheytv.up.railway.app/manifest.json
```

## Tech Stack

- Python / FastAPI
- MovieBox API
- TMDB API for metadata
- RPDB for rating posters
